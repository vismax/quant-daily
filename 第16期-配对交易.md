# 量化学习第16期：配对交易（Pairs Trading）原理与实践

## 📌 一句话概括
配对交易是一种统计套利策略，通过寻找价格走势相近的股票对，当它们价格偏离历史均衡时做多弱势股、做空强势股，等待价格回归均衡时获利。

## 📖 概念详解
配对交易（Pairs Trading）是一种市场中性策略，其核心思想基于均值回归理论。具体来说，配对交易是从市场上找出历史股价走势相近的两只或多只股票，当它们之间的价格差或比率偏离历史均值时，进行相反方向的交易操作。当配对股票价格偏离过大时，做空相对强势的股票，同时做多相对弱势的股票，当两者股价重返均衡值时平仓获利。

配对交易的理论基础是统计套利，它依赖于金融时间序列的协整性。两只股票如果存在长期均衡关系，即使短期内价格出现偏离，也会趋向于回到均衡状态。这种策略的优势在于市场中性，即无论市场整体上涨还是下跌，只要配对股票之间的相对价格关系发生变化，就有机会获利。

在实际操作中，配对交易需要经过严格的统计检验，包括相关性分析、协整检验、平稳性测试等。常用的配对方法包括距离法、协整法、聚类分析等。交易信号通常基于价差的标准化分数（如z-score），当价差偏离均值超过一定阈值时产生交易信号。

## 📐 数学公式

### 协整检验（Engle-Granger两步法）

**第一步：建立回归模型**
$$Y_t = \alpha + \beta X_t + \epsilon_t$$

**第二步：检验残差平稳性**
$$\epsilon_t = Y_t - \alpha - \beta X_t$$

残差需要通过ADF检验，如果残差是平稳的，则表明两序列存在协整关系。

### 价差计算公式
$$Spread_t = Price_A_t - \beta \times Price_B_t$$

### 标准化价差（Z-Score）
$$Z-Score_t = \frac{Spread_t - \mu_{Spread}}{\sigma_{Spread}}$$

其中：
- $\mu_{Spread}$ 是价差的均值
- $\sigma_{Spread}$ 是价差的标准差

### 交易信号阈值
- 买入信号：$Z-Score < -1.0$（价差低于均值1个标准差）
- 卖出信号：$Z-Score > 1.0$（价差高于均值1个标准差）
- 平仓信号：$Z-Score$ 接近0（价差回归均值）

## 🐍 Python 代码实战

```python
import numpy as np
import pandas as pd
import yfinance as yf
from statsmodels.tsa.stattools import coint, adfuller
from sklearn.preprocessing import MinMaxScaler
import matplotlib.pyplot as plt

class PairsTradingStrategy:
    def __init__(self, ticker1, ticker2, window=60, zscore_threshold=1.0):
        """
        初始化配对交易策略
        
        参数:
        ticker1, ticker2: 股票代码
        window: 滚动窗口大小
        zscore_threshold: z-score阈值
        """
        self.ticker1 = ticker1
        self.ticker2 = ticker2
        self.window = window
        self.zscore_threshold = zscore_threshold
        
    def download_data(self, start_date, end_date):
        """下载股票数据"""
        data1 = yf.download(self.ticker1, start=start_date, end=end_date)
        data2 = yf.download(self.ticker2, start=start_date, end=end_date)
        
        # 确保数据对齐
        data = pd.DataFrame({
            f'{self.ticker1}': data1['Adj Close'],
            f'{self.ticker2}': data2['Adj Close']
        })
        
        return data.dropna()
    
    def test_cointegration(self, data):
        """检验协整关系"""
        score, pvalue, _ = coint(data.iloc[:, 0], data.iloc[:, 1])
        return pvalue < 0.05  # p值小于0.05认为存在协整关系
    
    def calculate_spread(self, data):
        """计算价差"""
        # 使用线性回归计算对冲比率
        from sklearn.linear_model import LinearRegression
        
        X = data.iloc[:, 1].values.reshape(-1, 1)
        y = data.iloc[:, 0].values.reshape(-1, 1)
        
        model = LinearRegression()
        model.fit(X, y)
        hedge_ratio = model.coef_[0][0]
        
        # 计算价差
        spread = data.iloc[:, 0] - hedge_ratio * data.iloc[:, 1]
        
        return spread, hedge_ratio
    
    def calculate_zscore(self, spread):
        """计算z-score"""
        rolling_mean = spread.rolling(window=self.window).mean()
        rolling_std = spread.rolling(window=self.window).std()
        
        zscore = (spread - rolling_mean) / rolling_std
        return zscore
    
    def generate_signals(self, zscore):
        """生成交易信号"""
        signals = pd.DataFrame(index=zscore.index)
        
        # 买入信号（z-score < -threshold）
        signals['buy'] = np.where(zscore < -self.zscore_threshold, 1, 0)
        
        # 卖出信号（z-score > threshold）
        signals['sell'] = np.where(zscore > self.zscore_threshold, -1, 0)
        
        # 组合信号
        signals['signal'] = signals['buy'] + signals['sell']
        
        # 计算仓位变化
        signals['position'] = signals['signal'].diff()
        
        return signals
    
    def backtest(self, data, signals):
        """回测策略"""
        portfolio = pd.DataFrame(index=data.index)
        
        # 计算每日收益率
        returns = data.pct_change()
        
        # 假设等权重配置
        portfolio['return'] = returns.iloc[:, 0] * (signals['signal'] > 0).shift(1) + \
                            returns.iloc[:, 1] * (signals['signal'] < 0).shift(1)
        
        # 计算累计收益
        portfolio['cumulative_return'] = (1 + portfolio['return']).cumprod()
        
        return portfolio
    
    def plot_results(self, data, spread, zscore, signals):
        """绘制结果图表"""
        fig, (ax1, ax2, ax3) = plt.subplots(3, 1, figsize=(12, 10))
        
        # 价格走势
        ax1.plot(data.iloc[:, 0], label=f'{self.ticker1}', color='blue')
        ax1.plot(data.iloc[:, 1], label=f'{self.ticker2}', color='red')
        ax1.set_title('Stock Prices')
        ax1.legend()
        
        # 价差
        ax2.plot(spread, label='Spread', color='green')
        ax2.axhline(y=0, color='black', linestyle='--')
        ax2.set_title('Price Spread')
        ax2.legend()
        
        # Z-Score和交易信号
        ax3.plot(zscore, label='Z-Score', color='purple')
        ax3.axhline(y=self.zscore_threshold, color='red', linestyle='--', alpha=0.7)
        ax3.axhline(y=-self.zscore_threshold, color='red', linestyle='--', alpha=0.7)
        ax3.axhline(y=0, color='black', linestyle='--', alpha=0.5)
        
        # 标记交易信号
        buy_signals = signals[signals['position'] == 1].index
        sell_signals = signals[signals['position'] == -1].index
        
        ax3.scatter(buy_signals, zscore.loc[buy_signals], color='green', marker='^', s=100, label='Buy')
        ax3.scatter(sell_signals, zscore.loc[sell_signals], color='red', marker='v', s=100, label='Sell')
        
        ax3.set_title('Z-Score and Trading Signals')
        ax3.legend()
        
        plt.tight_layout()
        plt.show()

# 使用示例
if __name__ == "__main__":
    # 初始化策略
    strategy = PairsTradingStrategy('AAPL', 'MSFT', window=60, zscore_threshold=1.0)
    
    # 下载历史数据
    data = strategy.download_data('2023-01-01', '2024-12-31')
    
    # 检验协整关系
    is_cointegrated = strategy.test_cointegration(data)
    print(f"协整检验结果: {'存在协整关系' if is_cointegrated else '不存在协整关系'}")
    
    if is_cointegrated:
        # 计算价差
        spread, hedge_ratio = strategy.calculate_spread(data)
        
        # 计算z-score
        zscore = strategy.calculate_zscore(spread)
        
        # 生成交易信号
        signals = strategy.generate_signals(zscore)
        
        # 回测
        portfolio = strategy.backtest(data, signals)
        
        # 计算性能指标
        total_return = portfolio['cumulative_return'].iloc[-1] - 1
        sharpe_ratio = np.sqrt(252) * portfolio['return'].mean() / portfolio['return'].std()
        
        print(f"总收益率: {total_return:.2%}")
        print(f"夏普比率: {sharpe_ratio:.2f}")
        
        # 绘制结果
        strategy.plot_results(data, spread, zscore, signals)
    else:
        print("股票对不存在协整关系，不适合配对交易")
```

## 💡 实战应用

配对交易在实际应用中需要注意以下几个方面：

**1. 股票对选择**
- 同行业股票（如科技股、银行股等）
- 相似市值和流动性
- 历史相关性高（相关系数>0.7）
- 通过聚类算法筛选潜在配对

**2. 参数优化**
- 滚动窗口大小（通常30-90天）
- z-score阈值（1.0-2.0之间）
- 交易成本考虑
- 止盈止损设置

**3. 风险管理**
- 设置最大持仓比例
- 分散多个配对降低风险
- 定期重新检验协整关系
- 市场异常情况下的应急机制

**4. 实盘注意事项**
- 考虑滑点和交易成本
- 流动性风险控制
- 信息泄露风险防范
- 监管合规要求

## ⚠️ 常见陷阱

**1. 过度拟合陷阱**
- 过度优化历史参数导致未来表现不佳
- 解决方案：使用滚动回测和样本外验证

**2. 协整关系失效**
- 市场结构变化导致历史协整关系破裂
- 解决方案：定期重新检验协整性，设置失效退出机制

**3. 流动性风险**
- 配对股票流动性不足导致无法及时建仓平仓
- 解决方案：优先选择高流动性股票，设置最小交易量

**4. 交易成本侵蚀**
- 频繁交易导致手续费和滑点成本过高
- 解决方案：考虑交易成本，提高信号阈值减少交易频率

**5. 黑天鹅事件**
- 极端市场情况下配对关系失效
- 解决方案：设置最大损失限制，分散投资
