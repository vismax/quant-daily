## 📌 一句话概括

布林带通过移动均线 ± 标准差构建动态通道，让价格波动「可视化」，是量化交易中最直观的波动率工具。

---

## 📖 概念详解

### 什么是布林带？

布林带（Bollinger Bands）由 John Bollinger 在 1980 年代提出，由三条轨道组成：

- **中轨（MB）**：通常为 20 日简单移动平均线（SMA），代表价格的趋势方向
- **上轨（UB）**：中轨 + 2 倍标准差，代表价格波动的上边界
- **下轨（LB）**：中轨 - 2 倍标准差，代表价格波动的下边界

标准差的本质是度量价格偏离均值的程度。当市场波动加大时，带宽会自动扩张；当市场趋于平静时，带宽会收缩。这种「自适应」特性让布林带在震荡市和趋势市中都能提供有价值的信息。

### 布林带的物理直觉

想象一根橡皮筋：平静时价格围绕中轨小幅波动，橡皮筋松弛（带宽窄）；剧烈波动时价格远离中轨，橡皮筋绷紧（带宽宽）。统计学上，在正态分布假设下，约 95.4% 的价格会落在上下轨之间——但金融市场并非完美正态，所以价格偶尔会突破轨道，这本身就构成了交易信号。

### 带宽（BandWidth）与 %B

布林带还衍生出两个重要指标：

- **带宽（BW）** = (上轨 - 下轨) / 中轨 × 100，衡量当前波动率水平
- **%B** = (价格 - 下轨) / (上轨 - 下轨)，衡量价格在通道中的相对位置（0 = 在下轨，1 = 在上轨）

---

## 📐 数学公式

### 标准布林带

$$
\text{MB}_t = \text{SMA}(C, N) = \frac{1}{N} \sum_{i=0}^{N-1} C_{t-i}
$$

$$
\text{UB}_t = \text{MB}_t + k \cdot \sigma_t
$$

$$
\text{LB}_t = \text{MB}_t - k \cdot \sigma_t
$$

其中：
- $C_t$ 为第 t 期收盘价
- $N$ 为周期（默认 20）
- $k$ 为标准差倍数（默认 2）
- $\sigma_t = \sqrt{\frac{1}{N-1} \sum_{i=0}^{N-1} (C_{t-i} - \text{MB}_t)^2}$

### 带宽与 %B

$$
\text{BW}_t = \frac{\text{UB}_t - \text{LB}_t}{\text{MB}_t} \times 100
$$

$$
\%\text{B}_t = \frac{C_t - \text{LB}_t}{\text{UB}_t - \text{LB}_t}
$$

## 🐍 Python 代码实战

```python
"""
布林带计算与可视化 — 量化学习第05期
依赖：pip install pandas numpy matplotlib akshare
"""
import akshare as ak
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# ============ 1. 获取数据 ============
df = ak.stock_zh_a_hist(symbol="000001", period="daily",
                         start_date="20250901", end_date="20260325")
df.columns = ['日期', '开盘', '收盘', '最高', '最低', '成交量', '成交额', '振幅', '涨跌幅', '涨跌额', '换手率']
df['日期'] = pd.to_datetime(df['日期'])
df.set_index('日期', inplace=True)

# ============ 2. 计算布林带 ============
N = 20       # 周期
K = 2        # 标准差倍数

df['中轨'] = df['收盘'].rolling(window=N).mean()
df['标准差'] = df['收盘'].rolling(window=N).std()
df['上轨'] = df['中轨'] + K * df['标准差']
df['下轨'] = df['中轨'] - K * df['标准差']

# 衍生指标
df['带宽'] = (df['上轨'] - df['下轨']) / df['中轨'] * 100
df['%B'] = (df['收盘'] - df['下轨']) / (df['上轨'] - df['下轨'])

# ============ 3. 简单均值回归策略信号 ============
# 价格跌破下轨 → 买入信号；价格突破上轨 → 卖出信号
df['信号'] = 0
df.loc[df['收盘'] <= df['下轨'], '信号'] = 1    # 买入
df.loc[df['收盘'] >= df['上轨'], '信号'] = -1   # 卖出

# ============ 4. 可视化 ============
plt.rcParams['font.sans-serif'] = ['SimHei']
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 8), height_ratios=[3, 1],
                                sharex=True, gridspec_kw={'hspace': 0.05})

# 上半部分：K线 + 布林带
ax1.plot(df.index, df['收盘'], label='收盘价', color='#1f77b4', linewidth=1)
ax1.plot(df.index, df['中轨'], label=f'中轨 SMA({N})', color='gray', linewidth=1)
ax1.fill_between(df.index, df['上轨'], df['下轨'], alpha=0.15, color='steelblue', label='布林带通道')
ax1.plot(df.index, df['上轨'], '--', color='red', linewidth=0.8)
ax1.plot(df.index, df['下轨'], '--', color='green', linewidth=0.8)

# 标注买卖信号
buy_signals = df[df['信号'] == 1]
sell_signals = df[df['信号'] == -1]
ax1.scatter(buy_signals.index, buy_signals['收盘'], marker='^', color='red', s=80, label='买入信号', zorder=5)
ax1.scatter(sell_signals.index, sell_signals['收盘'], marker='v', color='green', s=80, label='卖出信号', zorder=5)

ax1.set_title('布林带策略 — 平安银行 (000001)', fontsize=14)
ax1.legend(loc='upper left', fontsize=9)
ax1.grid(alpha=0.3)

# 下半部分：带宽
ax2.fill_between(df.index, df['带宽'], alpha=0.4, color='orange')
ax2.set_ylabel('带宽 (%)', fontsize=10)
ax2.set_title('带宽（波动率指标）', fontsize=11)
ax2.grid(alpha=0.3)

plt.tight_layout()
plt.savefig('bollinger_bands.png', dpi=150, bbox_inches='tight')
plt.show()

print("=== 最近5日数据 ===")
print(df[['收盘', '中轨', '上轨', '下轨', '%B', '带宽']].tail())
```

## 💡 实战应用

### 1. 均值回归策略（震荡市）
价格触及下轨时买入，触及上轨时卖出。这是最经典的布林带策略，适合横盘震荡市场。配合 %B 指标可以更精确判断——%B < 0 时价格已跌破下轨，%B > 1 时已突破上轨。

### 2. 突破策略（趋势市）
价格沿上轨运行时持有做多（趋势延续），而非逢高卖出。判断趋势的关键：连续多日收盘价在布林带上半部分 → 强趋势；反之亦然。

### 3. 布林带收缩（Squeeze）— 变盘信号
当带宽降至历史低位时，意味着波动率被极度压缩，市场即将选择方向（暴涨或暴跌）。John Bollinger 本人称这是布林带最有价值的信号之一。

### 4. W 底与 M 顶识别
布林带可以帮助识别价格形态：
- **W 底**：价格两次触及下轨但未跌破中轨，第二次低点高于第一次 → 看涨
- **M 顶**：价格两次触及上轨但未能突破，第二次高点低于第一次 → 看跌

---

## ⚠️ 常见陷阱

1. **震荡市 vs 趋势市混淆**：均值回归策略在强趋势中会被反复打脸。价格沿上轨持续运行时不断「卖出」会错过大行情。务必先判断市场状态。
2. **参数迷信**：20 日 + 2 倍标准差只是默认值，不是万能公式。不同品种、不同周期需要不同参数。建议用历史数据优化，但避免过度拟合。
3. **只看布林带**：单独使用任何指标都有盲区。建议结合成交量、RSI 等指标验证信号。布林带 + RSI < 30 + 放量 → 更可靠的买入信号。
4. **忽略带宽变化**：带宽扩张后的首次回归往往不可靠。真正有意义的是带宽从极窄处开始扩张的时刻。
