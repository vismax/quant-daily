## 📌 一句话概括
统计学是量化交易的基础，相关性、协方差和正态分布是理解资产关系和风险管理的核心工具。

## 📖 概念详解

### 相关性（Correlation）
相关性衡量两个变量之间线性关系的强度和方向，取值范围在-1到1之间。当相关系数为正时，表示两个变量同向变动；为负时表示反向变动；为0时表示无线性关系。在量化交易中，相关性分析用于识别资产间的联动性，是构建投资组合和风险管理的重要基础。

### 协方差（Covariance）
协方差衡量两个变量共同变化的程度，是相关性的分子部分。与不同量纲的相关系数不同，协方差的数值取决于变量的单位，因此其绝对值没有直接的经济意义。但在投资组合理论中，协方差矩阵是计算资产风险贡献和配置权重的关键输入参数。

### 正态分布（Normal Distribution）
正态分布又称高斯分布，是统计学中最重要的一种概率分布。许多金融资产的收益率在短期内近似服从正态分布，这使得正态分布成为量化模型的重要假设基础。然而，实际市场中极端事件（黑天鹅）的出现频率往往高于正态分布预测，这也是风险管理需要考虑的重要因素。

## 📐 数学公式

### 相关系数公式
```
ρ(X,Y) = cov(X,Y) / (σ_X * σ_Y)
```
其中：
- ρ(X,Y) 是X和Y的相关系数
- cov(X,Y) 是X和Y的协方差
- σ_X 和 σ_Y 分别是X和Y的标准差

### 协方差公式
```
cov(X,Y) = E[(X - μ_X)(Y - μ_Y)]
```
其中：
- E[] 表示期望值
- μ_X 和 μ_Y 分别是X和Y的均值

### 正态分布概率密度函数
```
f(x) = (1 / (σ√(2π))) * e^(-(x-μ)²/(2σ²))
```
其中：
- μ 是均值
- σ 是标准差
- π 是圆周率

## 🐍 Python 代码实战

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from scipy import stats

# 生成示例数据
np.random.seed(42)
n_samples = 1000

# 模拟两个相关资产的价格
returns_a = np.random.normal(0.001, 0.02, n_samples)
returns_b = 0.7 * returns_a + np.random.normal(0.0005, 0.015, n_samples)

# 创建DataFrame
data = pd.DataFrame({
    'Asset_A': returns_a,
    'Asset_B': returns_b
})

# 计算统计量
correlation = data.corr().iloc[0, 1]
covariance = data.cov().iloc[0, 1]

print(f"相关系数: {correlation:.4f}")
print(f"协方差: {covariance:.6f}")

# 正态性检验
_, p_value_a = stats.normaltest(data['Asset_A'])
_, p_value_b = stats.normaltest(data['Asset_B'])

print(f"Asset_A正态性检验p值: {p_value_a:.4f}")
print(f"Asset_B正态性检验p值: {p_value_b:.4f}")

# 可视化
fig, axes = plt.subplots(2, 2, figsize=(12, 10))

# 散点图
axes[0, 0].scatter(data['Asset_A'], data['Asset_B'], alpha=0.6)
axes[0, 0].set_title('资产收益率散点图')
axes[0, 0].set_xlabel('Asset A 收益率')
axes[0, 0].set_ylabel('Asset B 收益率')

# 分布图
sns.histplot(data['Asset_A'], kde=True, ax=axes[0, 1])
axes[0, 1].set_title('Asset A 收益率分布')

sns.histplot(data['Asset_B'], kde=True, ax=axes[1, 1])
axes[1, 1].set_title('Asset B 收益率分布')

# Q-Q图
stats.probplot(data['Asset_A'], dist="norm", plot=axes[1, 0])
axes[1, 0].set_title('Asset A Q-Q图')

plt.tight_layout()
plt.savefig('/root/codespace/quant-daily/statistics_analysis.png')
plt.close()

print("分析图表已保存为 statistics_analysis.png")
```

## 💡 实战应用

### 1. 投资组合构建
利用相关性分析构建分散化投资组合，降低整体风险。负相关的资产组合可以有效对冲风险。

### 2. 配对交易策略
识别高相关性资产对，当价差偏离历史均值时进行套利交易。这种策略基于均值回归原理。

### 3. 风险管理
通过协方差矩阵计算投资组合的方差，进行风险预算管理和压力测试。

### 4. 因子模型构建
在多因子模型中，因子收益率之间的协方差矩阵是计算因子风险的重要基础。

## ⚠️ 常见陷阱

### 1. 相关系数陷阱
相关系数只衡量线性关系，无法捕捉非线性关系。两个变量可能存在很强的非线性关系但相关系数接近0。

### 2. 时间依赖性陷阱
金融时间序列数据往往存在时变相关性，历史相关性在市场环境变化时可能失效。

### 3. 正态分布假设陷阱
实际金融收益率往往呈现"尖峰厚尾"特征，比正态分布有更多的极端值，简单的正态假设会低估风险。

### 4. 多重共线性陷阱
在多元回归中，高度相关的自变量会导致系数估计不稳定，影响模型解释力。
