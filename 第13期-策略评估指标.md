# 第13期：策略评估指标（夏普比率、最大回撤等）

## 📌 一句话概括
策略评估指标是量化交易的核心，通过夏普比率、最大回撤等关键指标，科学衡量策略的风险调整收益与风险控制能力。

## 📖 概念详解

策略评估指标是量化交易中不可或缺的工具，它们帮助我们客观地衡量一个交易策略的表现。夏普比率作为最广泛接受的指标，量化了每承受一单位风险能获得多少超额收益，是区分优秀策略与普通策略的关键标准。最大回撤则告诉我们策略在最坏情况下可能面临的最大损失，是风险管理的重要依据。

这些指标共同构成了策略评估的完整体系。除了夏普比率和最大回撤，还包括信息比率、收益回撤比、胜率等多个维度。每个指标都有其特定的应用场景：夏普比率适合评估整体风险调整收益，最大回撤关注极端风险，信息比率衡量策略相对基准的表现，而收益回撤比则直接反映了风险调整后的收益水平。在实际应用中，需要综合使用多个指标，避免单一指标的局限性。

策略评估不仅是对历史表现的总结，更是对未来表现的预测。一个优秀的策略应该在多种市场环境下都保持相对稳定的表现，这就要求我们不仅要看绝对收益，更要关注风险调整后的收益。通过科学的评估体系，我们可以筛选出真正具有持续盈利能力的策略，避免陷入"幸存者偏差"的陷阱。

## 📐 数学公式

### 夏普比率 (Sharpe Ratio)
```
Sharpe Ratio = (策略年化收益率 - 无风险利率) / 策略收益率的标准差
```

### 最大回撤 (Maximum Drawdown)
```
最大回撤 = max(净值高点 - 净值低点) / 净值高点
```

### 信息比率 (Information Ratio)
```
信息比率 = (策略收益率 - 基准收益率) / 策略收益率与基准收益率差异的标准差
```

### 收益回撤比 (Return/Drawdown Ratio)
```
收益回撤比 = 年化收益率 / 最大回撤
```

## 🐍 Python 代码实战

```python
import numpy as np
import pandas as pd
import yfinance as yf
from datetime import datetime, timedelta

def calculate_sharpe_ratio(returns, risk_free_rate=0.02):
    """
    计算夏普比率
    :param returns: 收益率序列
    :param risk_free_rate: 无风险利率 (年化)
    :return: 夏普比率
    """
    if len(returns) == 0:
        return 0
    
    # 计算年化收益率
    annual_return = returns.mean() * 252
    
    # 计算年化波动率
    annual_volatility = returns.std() * np.sqrt(252)
    
    # 计算夏普比率
    sharpe_ratio = (annual_return - risk_free_rate) / annual_volatility
    
    return sharpe_ratio

def calculate_max_drawdown(returns):
    """
    计算最大回撤
    :param returns: 收益率序列
    :return: 最大回撤
    """
    if len(returns) == 0:
        return 0
    
    # 计算累积收益率
    cumulative_returns = (1 + returns).cumprod()
    
    # 计算累积收益率的最高点
    peak = cumulative_returns.expanding().max()
    
    # 计算回撤
    drawdown = (cumulative_returns - peak) / peak
    
    # 最大回撤
    max_drawdown = drawdown.min()
    
    return abs(max_drawdown)

def calculate_information_ratio(returns, benchmark_returns):
    """
    计算信息比率
    :param returns: 策略收益率
    :param benchmark_returns: 基准收益率
    :return: 信息比率
    """
    if len(returns) == 0 or len(benchmark_returns) == 0:
        return 0
    
    # 计算超额收益
    excess_returns = returns - benchmark_returns
    
    # 计算信息比率
    if excess_returns.std() != 0:
        information_ratio = excess_returns.mean() / excess_returns.std()
    else:
        information_ratio = 0
    
    return information_ratio

def calculate_performance_metrics(returns, benchmark_returns=None, risk_free_rate=0.02):
    """
    计算策略评估指标
    :param returns: 策略收益率序列
    :param benchmark_returns: 基准收益率序列 (可选)
    :param risk_free_rate: 无风险利率
    :return: 评估指标字典
    """
    metrics = {}
    
    # 基础指标
    metrics['total_return'] = (1 + returns).prod() - 1
    metrics['annual_return'] = returns.mean() * 252
    metrics['annual_volatility'] = returns.std() * np.sqrt(252)
    metrics['sharpe_ratio'] = calculate_sharpe_ratio(returns, risk_free_rate)
    metrics['max_drawdown'] = calculate_max_drawdown(returns)
    metrics['return_drawdown_ratio'] = metrics['annual_return'] / metrics['max_drawdown'] if metrics['max_drawdown'] != 0 else 0
    
    # 胜率相关指标
    win_rate = (returns > 0).mean()
    metrics['win_rate'] = win_rate
    metrics['win_loss_ratio'] = win_rate / (1 - win_rate) if win_rate < 1 else float('inf')
    
    # 如果提供了基准收益率，计算相对指标
    if benchmark_returns is not None:
        metrics['information_ratio'] = calculate_information_ratio(returns, benchmark_returns)
        metrics['beta'] = np.cov(returns, benchmark_returns)[0, 1] / np.var(benchmark_returns)
        metrics['alpha'] = metrics['annual_return'] - (risk_free_rate + metrics['beta'] * (benchmark_returns.mean() * 252 - risk_free_rate))
    
    return metrics

def generate_trading_signals(data, short_window=20, long_window=50):
    """
    生成简单的交易信号
    :param data: 价格数据
    :param short_window: 短期均线窗口
    :param long_window: 长期均线窗口
    :return: 交易信号
    """
    # 计算移动平均线
    data['MA_short'] = data['Close'].rolling(window=short_window).mean()
    data['MA_long'] = data['Close'].rolling(window=long_window).mean()
    
    # 生成交易信号
    data['Signal'] = 0
    data.loc[data['MA_short'] > data['MA_long'], 'Signal'] = 1
    data.loc[data['MA_short'] < data['MA_long'], 'Signal'] = -1
    
    # 计算收益率
    data['Returns'] = data['Close'].pct_change()
    data['Strategy_Returns'] = data['Signal'].shift(1) * data['Returns']
    
    return data

def backtest_strategy(ticker='AAPL', start_date='2023-01-01', end_date='2024-01-01'):
    """
    回测策略
    :param ticker: 股票代码
    :param start_date: 开始日期
    :param end_date: 结束日期
    :return: 回测结果
    """
    # 获取数据
    data = yf.download(ticker, start=start_date, end=end_date)
    
    # 生成交易信号
    data = generate_trading_signals(data)
    
    # 计算策略收益率
    strategy_returns = data['Strategy_Returns'].dropna()
    
    # 计算基准收益率
    benchmark_returns = data['Returns'].dropna()
    
    # 计算评估指标
    metrics = calculate_performance_metrics(strategy_returns, benchmark_returns)
    
    return {
        'data': data,
        'metrics': metrics,
        'strategy_returns': strategy_returns,
        'benchmark_returns': benchmark_returns
    }

def main():
    """
    主函数：演示策略评估指标的计算
    """
    print("=== 量化策略评估指标演示 ===\n")
    
    # 回测策略
    result = backtest_strategy('AAPL', '2023-01-01', '2024-01-01')
    
    # 打印结果
    print("策略评估指标：")
    for key, value in result['metrics'].items():
        print(f"{key}: {value:.4f}")
    
    print("\n=== 分析 ===")
    print(f"夏普比率: {result['metrics']['sharpe_ratio']:.4f}")
    print(f"最大回撤: {result['metrics']['max_drawdown']:.4f}")
    print(f"收益回撤比: {result['metrics']['return_drawdown_ratio']:.4f}")
    print(f"胜率: {result['metrics']['win_rate']:.4f}")
    
    # 判断策略质量
    sharpe = result['metrics']['sharpe_ratio']
    max_dd = result['metrics']['max_drawdown']
    
    print("\n=== 策略质量评估 ===")
    if sharpe > 1.5:
        print("✅ 优秀策略：夏普比率 > 1.5")
    elif sharpe > 1.0:
        print("✅ 良好策略：夏普比率 > 1.0")
    elif sharpe > 0.75:
        print("⚠️ 一般策略：夏普比率 > 0.75")
    else:
        print("❌ 不推荐策略：夏普比率 < 0.75")
    
    if max_dd < 0.15:
        print("✅ 风险控制良好：最大回撤 < 15%")
    elif max_dd < 0.25:
        print("⚠️ 风险控制一般：最大回撤 < 25%")
    else:
        print("❌ 风险控制较差：最大回撤 > 25%")

if __name__ == "__main__":
    main()
```

## 💡 实战应用

在实际量化交易中，策略评估指标的应用场景非常广泛：

1. **策略筛选**：通过夏普比率等指标筛选出真正有价值的策略，避免过度优化。通常要求夏普比率 > 1.0，最大回撤 < 20%。

2. **风险控制**：最大回撤直接关系到资金管理，帮助我们设定合理的仓位大小和止损点。例如，如果策略最大回撤为15%，那么单次交易的仓位不应超过总资金的6.67%（15% ÷ 2.25）。

3. **策略优化**：通过信息比率等指标，我们可以判断策略是否真正具有Alpha，还是仅仅承担了额外的风险。信息比率 > 0.5 通常被认为是有效的。

4. **组合管理**：在多策略组合中，通过相关性分析和风险调整收益，实现最优的资产配置，降低整体风险。

## ⚠️ 常见陷阱

1. **幸存者偏差**：只关注表现好的策略，忽略失败案例，导致对策略表现过于乐观。

2. **过度拟合**：在历史数据上表现完美，但在实际交易中表现糟糕。高夏普比率（>3.0）和高胜率（>90%）往往是过度拟合的信号。

3. **样本量不足**：交易次数过少（<30次）时，统计指标不可靠，容易产生假象。

4. **基准选择不当**：选择不合适的基准会导致信息比率等指标失真，无法准确衡量策略的相对表现。

5. **忽略极端风险**：只关注夏普比率等平均指标，忽视最大回撤等极端风险指标，可能导致灾难性损失。
