## 📌 一句话概括

成交量是市场的"温度计"，价格告诉你发生了什么，成交量告诉你背后有多少人参与——两者结合才能看清趋势的真伪。

---

## 📖 概念详解

### 什么是成交量？

成交量（Trading Volume）指在特定时间段内，某一金融资产成功交易的总数量。每一笔成交必然有买方和卖方，成交量反映的是市场的参与度和活跃程度。在量化交易中，成交量是最基础、最重要的数据维度之一，几乎所有策略都会直接或间接地使用它。

### 为什么成交量如此重要？

经典的量价关系可以用一句话概括：**价格是表象，成交量是动力。** 没有成交量配合的价格突破，往往是假突破。成交量放大说明资金在积极进出，成交量萎缩说明市场犹豫不决。在实战中，成交量常被用来：
- **确认趋势强度**：上涨放量 + 下跌缩量 = 健康上涨
- **识别反转信号**：量价背离（价格新高但成交量下降）暗示趋势可能反转
- **判断突破有效性**：突破关键位时放量 = 真突破，缩量 = 假突破

### 核心成交量指标

| 指标 | 全称 | 核心用途 |
|------|------|----------|
| OBV | On Balance Volume（能量潮） | 通过累积量预测趋势变化 |
| VWAP | Volume Weighted Average Price | 日内"公平价"，机构基准 |
| MFI | Money Flow Index（资金流量指数） | 量化版的 RSI，结合量价 |
| A/D | Accumulation/Distribution（累积派发） | 衡量资金流入流出 |
| Chaikin | Chaikin Money Flow（柴金资金流） | A/D 的振荡器版本 |

## 📐 数学公式

### OBV（能量潮指标）

OBV 由 Joe Granville 于 1963 年提出，核心思想是：**成交量变化领先于价格变化**。

$$
OBV_t = OBV_{t-1} + \begin{cases} Volume_t, & \text{if } Close_t > Close_{t-1} \\ 0, & \text{if } Close_t = Close_{t-1} \\ -Volume_t, & \text{if } Close_t < Close_{t-1} \end{cases}
$$

### VWAP（成交量加权平均价）

VWAP 是日内交易中最重要的基准价格之一，反映当日的"平均成交成本"。

$$
VWAP = \frac{\sum_{i=1}^{n} (High_i + Low_i + Close_i) / 3 \times Volume_i}{\sum_{i=1}^{n} Volume_i}
$$

### MFI（资金流量指数）

MFI 由 J. Welles Wilder 在 1978 年提出，是结合了成交量的 RSI，范围 0-100。

$$
\text{Typical Price} = \frac{High + Low + Close}{3}
$$

$$
\text{Raw Money Flow} = \text{Typical Price} \times Volume
$$

$$
MFI = 100 - \frac{100}{1 + \frac{\sum Positive\ Money\ Flow}{\sum Negative\ Money\ Flow}}
$$

## 🐍 Python 代码实战

```python
"""
量化学习第06期：成交量分析实战
包含 OBV、VWAP、MFI 的计算与简单策略演示
"""

import numpy as np
import pandas as pd
import yfinance as yf
import matplotlib.pyplot as plt

# ========== 1. 获取数据 ==========
print("正在获取数据...")
df = yf.download('AAPL', start='2025-01-01', end='2026-03-26', progress=False)
df.columns = df.columns.get_level_values(0)  # 扁平化多级列名
df = df.dropna()

# ========== 2. 计算 OBV ==========
def calculate_obv(df):
    """能量潮指标：根据收盘价涨跌累积成交量"""
    obv = [0]
    for i in range(1, len(df)):
        if df['Close'].iloc[i] > df['Close'].iloc[i-1]:
            obv.append(obv[-1] + df['Volume'].iloc[i])
        elif df['Close'].iloc[i] < df['Close'].iloc[i-1]:
            obv.append(obv[-1] - df['Volume'].iloc[i])
        else:
            obv.append(obv[-1])
    return pd.Series(obv, index=df.index)

# ========== 3. 计算 VWAP ==========
def calculate_vwap(df):
    """成交量加权平均价：机构日内基准价格"""
    typical_price = (df['High'] + df['Low'] + df['Close']) / 3
    return (typical_price * df['Volume']).cumsum() / df['Volume'].cumsum()

# ========== 4. 计算 MFI ==========
def calculate_mfi(df, period=14):
    """资金流量指数：结合成交量的RSI（0-100）"""
    typical_price = (df['High'] + df['Low'] + df['Close']) / 3
    raw_money_flow = typical_price * df['Volume']
    
    money_flow_diff = typical_price.diff()
    
    positive_flow = pd.Series(0.0, index=df.index)
    negative_flow = pd.Series(0.0, index=df.index)
    
    positive_flow[money_flow_diff > 0] = raw_money_flow[money_flow_diff > 0]
    negative_flow[money_flow_diff < 0] = raw_money_flow[money_flow_diff < 0]
    
    positive_mf = positive_flow.rolling(window=period).sum()
    negative_mf = negative_flow.rolling(window=period).sum()
    
    mfi = 100 - (100 / (1 + positive_mf / negative_mf.replace(0, np.nan)))
    return mfi

# ========== 5. 计算所有指标 ==========
df['OBV'] = calculate_obv(df)
df['VWAP'] = calculate_vwap(df)
df['MFI_14'] = calculate_mfi(df, 14)
df['OBV_MA20'] = df['OBV'].rolling(window=20).mean()

print(f"数据范围：{df.index[0].date()} ~ {df.index[-1].date()}，共 {len(df)} 个交易日")

# ========== 6. OBV + 价格背离检测 ==========
def detect_obv_divergence(df, lookback=50):
    """检测 OBV 与价格的顶背离 / 底背离"""
    recent = df.tail(lookback)
    signals = []
    
    price_high = recent['Close'].idxmax()
    price_low = recent['Close'].idxmin()
    obv_at_price_high = recent.loc[price_high, 'OBV']
    obv_at_price_low = recent.loc[price_low, 'OBV']
    
    # 顶背离：价格新高，OBV 没有新高
    if price_high > recent.index[lookback // 2]:
        obv_before = recent['OBV'].loc[:price_high].iloc[:-1].max() if len(recent.loc[:price_high]) > 1 else obv_at_price_high
        if obv_at_price_high < obv_before:
            signals.append(('顶背离', price_high))
    
    # 底背离：价格新低，OBV 没有新低
    if price_low > recent.index[lookback // 2]:
        obv_before = recent['OBV'].loc[:price_low].iloc[:-1].min() if len(recent.loc[:price_low]) > 1 else obv_at_price_low
        if obv_at_price_low > obv_before:
            signals.append(('底背离', price_low))
    
    return signals

divergences = detect_obv_divergence(df)
if divergences:
    for dtype, date in divergences:
        print(f"⚠️ 检测到 {dtype} 信号：{date.date()}")
else:
    print("✅ 近50日未检测到明显背离信号")

# ========== 7. VWAP 日内策略回测（示例） ==========
def vwap_strategy(df):
    """
    简单 VWAP 策略：
    - 价格在 VWAP 上方 → 做多信号
    - 价格跌破 VWAP → 做空/止损信号
    """
    df = df.copy()
    df['Signal'] = 0
    df.loc[df['Close'] > df['VWAP'], 'Signal'] = 1    # VWAP 上方做多
    df.loc[df['Close'] < df['VWAP'], 'Signal'] = -1   # VWAP 下方做空
    
    df['Returns'] = df['Close'].pct_change()
    df['Strategy_Returns'] = df['Signal'].shift(1) * df['Returns']
    df['Cumulative_Returns'] = (1 + df['Returns']).cumprod()
    df['Cumulative_Strategy'] = (1 + df['Strategy_Returns']).cumprod()
    
    total_return = (df['Cumulative_Strategy'].iloc[-1] - 1) * 100
    buy_hold_return = (df['Cumulative_Returns'].iloc[-1] - 1) * 100
    
    return total_return, buy_hold_return

strategy_return, buy_hold_return = vwap_strategy(df)
print(f"\n📊 回测结果：")
print(f"  VWAP 策略收益率：{strategy_return:.2f}%")
print(f"  买入持有收益率：{buy_hold_return:.2f}%")

# ========== 8. 可视化 ==========
fig, axes = plt.subplots(3, 1, figsize=(14, 12), sharex=True)

# 子图1：价格 + VWAP + 成交量
ax1 = axes[0]
ax1.plot(df.index, df['Close'], label='Close', color='black', linewidth=1)
ax1.plot(df.index, df['VWAP'], label='VWAP', color='orange', linewidth=1.5, linestyle='--')
ax1.set_ylabel('Price ($)')
ax1.set_title('AAPL Price + VWAP', fontsize=14, fontweight='bold')
ax1.legend(loc='upper left')
ax1.grid(True, alpha=0.3)

ax1_vol = ax1.twinx()
colors = ['red' if df['Close'].iloc[i] < df['Close'].iloc[i-1] else 'green' 
          for i in range(1, len(df))]
ax1_vol.bar(df.index[1:], df['Volume'].iloc[1:], color=colors, alpha=0.3, width=1)
ax1_vol.set_ylabel('Volume')

# 子图2：OBV
ax2 = axes[1]
ax2.plot(df.index, df['OBV'], label='OBV', color='blue', linewidth=1)
ax2.plot(df.index, df['OBV_MA20'], label='OBV MA20', color='red', linewidth=1, linestyle='--')
ax2.set_ylabel('OBV')
ax2.set_title('On Balance Volume (OBV)', fontsize=14, fontweight='bold')
ax2.legend(loc='upper left')
ax2.grid(True, alpha=0.3)

# 子图3：MFI
ax3 = axes[2]
ax3.plot(df.index, df['MFI_14'], label='MFI(14)', color='purple', linewidth=1)
ax3.axhline(y=80, color='red', linestyle='--', alpha=0.5, label='Overbought (80)')
ax3.axhline(y=20, color='green', linestyle='--', alpha=0.5, label='Oversold (20)')
ax3.fill_between(df.index, 80, df['MFI_14'], where=(df['MFI_14'] > 80), 
                  color='red', alpha=0.2)
ax3.fill_between(df.index, 20, df['MFI_14'], where=(df['MFI_14'] < 20), 
                  color='green', alpha=0.2)
ax3.set_ylabel('MFI')
ax3.set_xlabel('Date')
ax3.set_title('Money Flow Index (MFI)', fontsize=14, fontweight='bold')
ax3.legend(loc='upper left')
ax3.grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('/root/codespace/quant-daily/images/06_volume_analysis.png', dpi=150, bbox_inches='tight')
print("\n✅ 图表已保存到 images/06_volume_analysis.png")
plt.close()
```

## 💡 实战应用

**场景一：OBV 突破确认**

当价格突破关键阻力位时，如果 OBV 同步创出新高，说明"聪明钱"在推动突破，趋势更可靠。如果价格突破但 OBV 没有跟上，要警惕假突破。

**场景二：VWAP 作为日内交易的"公平价"**

机构交易员常以 VWAP 作为执行基准。价格在 VWAP 上方运行时偏多头，跌破 VWAP 则转弱。日内交易者可以在 VWAP 附近设置支撑/阻力参考位。

**场景三：MFI 超买超卖判断**

MFI > 80 为超买区，MFI < 20 为超卖区。但与 RSI 类似，在强趋势中 MFI 可能长时间处于极端区域，不能单纯依赖此信号反向操作。

---

## ⚠️ 常见陷阱

1. **成交量 ≠ 方向**：成交量放大不代表价格一定涨，只能说明市场活跃度提高。必须结合价格方向一起分析。
2. **外汇市场成交量特殊**：外汇没有中心化交易所，"成交量"数据通常是 tick volume（价格变动次数），与真实成交量有差距。
3. **低流动性市场噪音大**：在小市值股票或冷门品种上，成交量指标容易被少数几笔大单扭曲。
4. **VWAP 只适用于日内**：VWAP 是每日重置的指标，跨日使用没有意义。跨周期请使用 VWMA（成交量加权移动均线）。
5. **背离不等于立即反转**：OBV 顶背离出现后，价格可能继续上涨很长时间。背离只是预警，不是信号。
