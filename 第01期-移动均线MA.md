# 📚 量化学习第01期：移动均线（MA）— 量化入门第一课

> 📌 **一句话概括：** 移动均线就是"价格的记忆"，它把过去 N 天的收盘价取个平均，画成一条平滑的线，帮你过滤噪音、看清趋势。

---

## 📖 概念详解

### 什么是移动均线？

想象你每天记录一个数值，但数据上下跳动很厉害，看不清趋势。怎么办？**取平均值**。

比如今天、昨天、前天的收盘价分别是 10.2、10.5、10.1，3日均线就是 (10.2+10.5+10.1)/3 = 10.27。

"移动"的意思是：每天向前滚动一天，始终取最近 N 天的平均值。这样随着时间推移，你就得到一条平滑的曲线。

用程序员思维理解：**移动均线就是一个长度为 N 的滑动窗口，对窗口内的数据求均值，窗口每次向右滑动一步。**

### 为什么均线有用？

价格数据充满了"噪音"——随机波动、情绪交易、流动性冲击……这些噪音让人看不清方向。均线做的事情很简单但很强大：**降噪。**

就像你在代码里对传感器数据做低通滤波一样，均线就是金融数据的低通滤波器。高频噪声被过滤掉，留下的是信号（趋势）。

### 均线的本质是"延迟"

这是最重要的概念：**均线越长的，延迟越大。**

类比一下：
- 5日均线 = 快速响应，但噪音多（像 1ms 的 ping，反应快但不一定准）
- 120日均线 = 非常平滑，但严重滞后（像 500ms 的 ping，准但反应慢）

没有完美的均线周期，这是你做量化要面对的第一个 trade-off。

## 📐 数学公式

### 简单移动均线（SMA，Simple Moving Average）

```
SMA(N) = (P₁ + P₂ + P₃ + ... + Pₙ) / N
```

- `SMA(N)` = N 日简单移动均线
- `P₁` 到 `Pₙ` = 最近 N 天的收盘价（P₁ 是 N 天前，Pₙ 是今天）
- `N` = 周期参数（常用：5、10、20、60、120、250）

**特点：** 每个数据点的权重相同，都是 1/N。

### Python 等价写法

```python
# SMA 本质就是 pandas 的 rolling mean
sma_20 = df['close'].rolling(window=20).mean()
```

### 指数移动均线（EMA，Exponential Moving Average）

```
EMA(t) = α × P(t) + (1 - α) × EMA(t-1)
其中 α = 2 / (N + 1)
```

- `EMA(t)` = 今天的 EMA 值
- `P(t)` = 今天的收盘价
- `EMA(t-1)` = 昨天的 EMA 值
- `α` = 平滑因子（也叫衰减系数），N 越大 α 越小，对近期价格越不敏感

**特点：** 越近的价格权重越大。类比：SMA 是"所有历史记录同等重要"，EMA 是"昨天发生的事比上周更重要"。

### Python 等价写法

```python
# 方法一：pandas ewm
ema_20 = df['close'].ewm(span=20, adjust=False).mean()

# 方法二：手工实现（帮助理解原理）
alpha = 2 / (20 + 1)
ema = [df['close'].iloc[0]]  # 初始值取第一个收盘价
for price in df['close'].iloc[1:]:
    ema.append(alpha * price + (1 - alpha) * ema[-1])
```

## 🐍 Python 代码实战

下面用 `akshare` 获取真实 A 股数据，实现一个完整的双均线策略：

```python
"""
第01期：双均线策略（Golden Cross / Death Cross）
获取贵州茅台（600519）最近 2 年的日线数据
短期均线：MA20，长期均线：MA60
"""
import akshare as ak
import pandas as pd
import matplotlib.pyplot as plt

# ===== 1. 获取数据 =====
# akshare 是免费的 A 股数据源，pip install akshare
df = ak.stock_zh_a_hist(symbol="600519", period="daily",
                        start_date="20240101", end_date="20260321", adjust="qfq")
# qfq = 前复权，消除分红送股对价格的影响

df = df[["日期", "收盘", "开盘", "最高", "最低", "成交量"]]
df.columns = ["date", "close", "open", "high", "low", "volume"]
df["date"] = pd.to_datetime(df["date"])
df.set_index("date", inplace=True)
df = df.sort_index()

# ===== 2. 计算均线 =====
df["ma_short"] = df["close"].rolling(window=20).mean()   # 20日均线
df["ma_long"]  = df["close"].rolling(window=60).mean()   # 60日均线

# ===== 3. 生成交易信号 =====
# 金叉（买入）：短期均线上穿长期均线
# 死叉（卖出）：短期均线下穿长期均线
df["signal"] = 0
df.loc[df["ma_short"] > df["ma_long"], "signal"] = 1   # 持多头
df.loc[df["ma_short"] < df["ma_long"], "signal"] = -1  # 持空头（或空仓）

# 找交叉点（信号变化的位置）
df["cross"] = df["signal"].diff()
buy_signals  = df[df["cross"] == 2]   # 从 -1 变 1 = 金叉
sell_signals = df[df["cross"] == -2]  # 从 1 变 -1 = 死叉

print(f"共发现 {len(buy_signals)} 次金叉，{len(sell_signals)} 次死叉")
print(f"\n最近 3 次买入信号：")
print(buy_signals[["close", "ma_short", "ma_long"]].tail(3))

# ===== 4. 简单回测 =====
df["returns"] = df["close"].pct_change()  # 每日收益率
df["strategy_returns"] = df["signal"].shift(1) * df["returns"]  # 策略收益

total_return = (1 + df["strategy_returns"]).prod() - 1
buy_hold_return = df["close"].iloc[-1] / df["close"].iloc[0] - 1

print(f"\n===== 回测结果 =====")
print(f"策略累计收益：{total_return:.2%}")
print(f"买入持有收益：{buy_hold_return:.2%}")

# ===== 5. 可视化 =====
plt.figure(figsize=(14, 7))
plt.plot(df.index, df["close"], label="收盘价", color="black", alpha=0.5, linewidth=0.8)
plt.plot(df.index, df["ma_short"], label="MA20", color="blue", linewidth=1.2)
plt.plot(df.index, df["ma_long"],  label="MA60", color="red",  linewidth=1.2)

# 标记买卖点
plt.scatter(buy_signals.index, buy_signals["close"],
            marker="^", color="green", s=100, label="买入（金叉）", zorder=5)
plt.scatter(sell_signals.index, sell_signals["close"],
            marker="v", color="red", s=100, label="卖出（死叉）", zorder=5)

plt.title("贵州茅台（600519）双均线策略", fontsize=14)
plt.xlabel("日期", fontsize=12)
plt.ylabel("价格（元）", fontsize=12)
plt.legend(fontsize=10)
plt.grid(alpha=0.3)
plt.tight_layout()
plt.savefig("ma_strategy.png", dpi=150)
plt.show()
print("图表已保存为 ma_strategy.png")
```

## 💡 实战应用

### 常用均线周期一览

| 周期 | 名称 | 用途 |
|------|------|------|
| MA5 | 周线 | 超短线，日内/隔日交易 |
| MA10 | 两周线 | 短线趋势 |
| MA20 | 月线 | 最常用的短期趋势线 |
| MA60 | 季线 | 中期趋势，机构关注度高 |
| MA120 | 半年线 | 长期趋势支撑/压力 |
| MA250 | 年线 | 牛熊分界线，跌破年线 = 技术性熊市 |

### 双均线策略的典型参数组合

- **日内交易：** MA5 / MA10
- **波段交易：** MA10 / MA30 或 MA20 / MA60
- **中线趋势：** MA30 / MA120
- **长线趋势：** MA60 / MA250

### 均线和其他指标的配合

- **均线 + 成交量：** 均线金叉 + 放量 = 信号更可靠
- **均线 + RSI：** 均线金叉但 RSI 已经 > 70 = 可能追高，要警惕
- **均线 + 布林带：** 价格在布林带下轨反弹 + 回到均线之上 = 更强的买入信号

## ⚠️ 常见陷阱

### 1. 震荡市的"反复打脸"
均线策略在**趋势行情**中表现优秀，但在**横盘震荡**中会频繁产生假信号——金叉买入马上死叉卖出，来回被割。

**对策：** 加一个"趋势过滤器"，比如 ADX 指标 > 25 时才交易，或者只在均线的斜率向上时才考虑金叉信号。

### 2. 参数过拟合
回测时发现 MA17/MA43 组合收益最高，实盘一跑就亏——因为你把参数"拟合"到了历史数据上。

**对策：** 选择"经典"参数（MA20/MA60），或者用 Walk-Forward 分析验证参数稳定性。不要为了一点回测收益去优化参数。

### 3. 忽略交易成本
双均线策略在震荡市可能一年交易 50 次，每次 0.15% 的手续费 + 滑点，累计成本可能吃掉全部收益。

**对策：** 回测时一定要加入手续费计算（至少万三佣金 + 千一滑点）。

### 4. 滞后性的本质局限
均线是"看着后视镜开车"——它永远基于过去的价格，对突发事件反应慢。

**对策：** 没有完美的解决方案，但可以结合更快响应的指标（如价格突破）来辅助判断。
