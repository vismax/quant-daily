## 📌 一句话概括
从数据获取到策略回测再到评估分析的完整流程，构建你的第一个量化交易策略。

## 📖 概念详解

量化交易策略的开发遵循系统化的流程，从数据获取开始，到策略构建、回测验证，最后进行评估优化。一个完整的量化策略包括交易模型、条件因子、买卖执行和监控优化四个核心部分。其中，交易模型是整个量化策略的核心，而回测则是验证策略有效性的关键环节。

回测是指将交易策略应用于历史市场数据，模拟策略在过去的表现。通过回测，我们可以评估策略的潜在盈利能力、风险水平和回撤情况，从而在投入真实资金之前对策略进行充分验证。这个过程不仅能帮助我们发现策略的优势和不足，还能避免在实际交易中可能出现的重大损失。

完整的策略开发流程需要考虑多个方面：数据的质量和完整性、策略逻辑的严谨性、回测方法的科学性，以及评估指标的全面性。只有经过充分回测和严格评估的策略，才能在实际市场中获得稳定的表现。

## 📐 数学公式

### 回测核心指标计算

```
# 年化收益率
年化收益率 = (总收益率 + 1)^(365/交易天数) - 1

# 最大回撤
最大回撤 = max(累计最高价 - 当前价格) / 累计最高价

# 夏普比率
夏普比率 = (年化收益率 - 无风险利率) / 年化波动率

# 胜率
胜率 = 盈利交易次数 / 总交易次数

# 盈亏比
盈亏比 = 平均盈利金额 / 平均亏损金额
```

### 双均线策略信号

```
# 简单移动均线
SMA(n) = (P₁ + P₂ + ... + Pₙ) / n

# 指数移动均线
EMA(n) = P × α + EMA(n-1) × (1-α)
其中 α = 2 / (n + 1)

# 交易信号
金叉：短期EMA > 长期EMA，买入信号
死叉：短期EMA < 长期EMA，卖出信号
```

## 🐍 Python 代码实战

```python
import pandas as pd
import numpy as np
import akshare as ak
import matplotlib.pyplot as plt
from datetime import datetime, timedelta

def get_stock_data(symbol='000001', start_date='2023-01-01', end_date='2024-01-01'):
    """获取股票历史数据"""
    try:
        df = ak.stock_zh_a_hist(symbol=symbol, period="daily", 
                                start_date=start_date, end_date=end_date, 
                                adjust="hfq")
        df['日期'] = pd.to_datetime(df['日期'])
        df.set_index('日期', inplace=True)
        df['收盘价'] = df['收盘'].astype(float)
        return df[['收盘价']]
    except Exception as e:
        print(f"数据获取失败: {e}")
        return None

def calculate_ema(data, period):
    """计算指数移动均线"""
    return data['收盘价'].ewm(span=period, adjust=False).mean()

def backtest_strategy(df, short_period=5, long_period=20):
    """双均线策略回测"""
    # 计算均线
    df['短期EMA'] = calculate_ema(df, short_period)
    df['长期EMA'] = calculate_ema(df, long_period)
    
    # 生成交易信号
    df['信号'] = 0
    df.loc[df['短期EMA'] > df['长期EMA'], '信号'] = 1  # 买入信号
    df.loc[df['短期EMA'] < df['长期EMA'], '信号'] = -1  # 卖出信号
    
    # 计算持仓
    df['持仓'] = df['信号'].shift(1)
    
    # 计算收益率
    df['收益率'] = df['收盘价'].pct_change()
    df['策略收益率'] = df['持仓'] * df['收益率']
    
    # 计算累计收益
    df['累计收益'] = (1 + df['策略收益率']).cumprod()
    
    return df

def calculate_metrics(df):
    """计算策略评估指标"""
    total_return = df['累计收益'].iloc[-1] - 1
    
    # 计算最大回撤
    cumulative_max = df['累计收益'].cummax()
    drawdown = (cumulative_max - df['累计收益']) / cumulative_max
    max_drawdown = drawdown.max()
    
    # 计算年化收益率
    days = len(df)
    annual_return = (df['累计收益'].iloc[-1] ** (365/days)) - 1
    
    # 计算夏普比率
    daily_returns = df['策略收益率'].dropna()
    sharpe_ratio = np.sqrt(252) * daily_returns.mean() / daily_returns.std()
    
    return {
        '总收益率': f"{total_return:.2%}",
        '最大回撤': f"{max_drawdown:.2%}",
        '年化收益率': f"{annual_return:.2%}",
        '夏普比率': f"{sharpe_ratio:.2f}"
    }

def plot_results(df):
    """绘制回测结果"""
    plt.figure(figsize=(12, 8))
    
    # 绘制价格和均线
    plt.subplot(2, 1, 1)
    plt.plot(df['收盘价'], label='收盘价', alpha=0.7)
    plt.plot(df['短期EMA'], label=f'{5}日EMA', alpha=0.8)
    plt.plot(df['长期EMA'], label=f'{20}日EMA', alpha=0.8)
    
    # 标记买卖点
    buy_signals = df[df['信号'] == 1]
    sell_signals = df[df['信号'] == -1]
    
    plt.scatter(buy_signals.index, buy_signals['收盘价'], 
                color='red', marker='^', s=100, label='买入')
    plt.scatter(sell_signals.index, sell_signals['收盘价'], 
                color='green', marker='v', s=100, label='卖出')
    
    plt.title('双均线策略回测结果')
    plt.legend()
    plt.grid(True)
    
    # 绘制收益曲线
    plt.subplot(2, 1, 2)
    plt.plot(df['累计收益'], label='策略收益', color='blue')
    plt.plot(df['收盘价'] / df['收盘价'].iloc[0], label='买入持有', color='orange', alpha=0.7)
    plt.title('累计收益对比')
    plt.legend()
    plt.grid(True)
    
    plt.tight_layout()
    plt.savefig('/tmp/backtest_results.png', dpi=300, bbox_inches='tight')
    plt.close()

def main():
    """主函数：完整策略回测流程"""
    print("=== 第一个完整策略：从数据到回测到评估 ===")
    
    # 1. 获取数据
    print("1. 获取股票数据...")
    df = get_stock_data('000001', '2023-01-01', '2024-01-01')
    if df is None:
        return
    
    print(f"数据获取完成，共 {len(df)} 个交易日")
    
    # 2. 执行回测
    print("2. 执行策略回测...")
    df_backtest = backtest_strategy(df)
    
    # 3. 计算指标
    print("3. 计算策略指标...")
    metrics = calculate_metrics(df_backtest)
    
    print("\n=== 回测结果 ===")
    for key, value in metrics.items():
        print(f"{key}: {value}")
    
    # 4. 绘制图表
    print("4. 生成回测图表...")
    plot_results(df_backtest)
    print("图表已保存至 /tmp/backtest_results.png")
    
    # 5. 保存结果
    print("5. 保存回测结果...")
    result_df = pd.DataFrame([metrics])
    result_df.to_csv('/tmp/backtest_metrics.csv', index=False)
    print("结果已保存至 /tmp/backtest_metrics.csv")

if __name__ == "__main__":
    main()
```

## 💡 实战应用

1. **数据获取与清洗**：使用AKShare获取真实的历史数据，确保数据的准确性和完整性
2. **策略构建**：基于双均线交叉原理，建立明确的买卖规则
3. **回测执行**：将策略应用到历史数据中，记录每次交易决策
4. **结果分析**：通过多个指标综合评估策略表现，包括收益、风险、稳定性等
5. **优化迭代**：根据回测结果调整参数，寻找最优策略配置

## ⚠️ 常见陷阱

1. **过拟合陷阱**：过度优化历史数据，导致策略在实盘中表现不佳
2. **未来函数**：在回测中使用未来数据，造成策略表现虚高
3. **幸存者偏差**：只研究当前存在的股票，忽略了退市股票的影响
4. **交易成本忽略**：未考虑手续费、滑点等交易成本，高估策略收益
5. **样本不足**：使用过短的历史数据进行回测，结果缺乏统计显著性
